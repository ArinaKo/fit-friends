Параметры:

  • image - имя файла изображения,
  • imagePath - путь к файлу изображения (по умолчанию - "content/thumbnails"),

  • imageAlt - alt для изображения (по умолчанию - пустой),
  • title - название зала,
  • titleLevel - уровень заголовка для названия зала (по умолчанию - h3),
  • favourite - избранное (по умолчанию - false),
  • verified - проверенный (по умолчанию - false),
  • location - локация зала,
  • text - описание зала,
  • btnRef - ссылка на страницу зала
